# Smart Clinic Booking System - Backend API

A robust and secure RESTful API built with **Node.js**, **Express.js**, and **MongoDB** using Clean Architecture.

## Core Features
- **Role-Based Access Control (RBAC):** Distinct permissions for Patients, Doctors, and Admins.
- **Advanced Booking Logic:** Prevents double-booking by dynamically checking doctor availability.
- **Doctor Dashboard:** Manage availability slots, view, confirm, or cancel appointments.
- **Admin Panel:** Full CRUD operations to monitor and manage all system data.
